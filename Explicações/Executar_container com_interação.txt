Executar container com interação
●Podemos rodar um container e deixá-lo executando no terminal;
●Vamos utilizar a flag -it;
●Desta maneira podemos executar comandos disponíveis no container
que estamos utilizando o comando run;
●
Podemos utilizar a imagem do ubuntu para isso!
Ex.:
desenvolvedor@desenvolvedor-ThinkPad-T440p:~/docker-compose-test$ docker run -it node
